# API Researcher Skill

Comprehensive API research and documentation skill that investigates external APIs, libraries, and services, producing standardized documentation for integration.

## When to Use

- Researching a new REST API, GraphQL API, or cloud service
- Documenting authentication methods for external services
- Creating integration guides before implementation
- Understanding Python libraries or npm packages

## Features

| Feature | Description |
|---------|-------------|
| Multi-source research | Official docs, GitHub, web search |
| Standardized output | 5 document types per API |
| Collaboration patterns | Works with orchestrator and other agents |
| Scenario coverage | REST, GraphQL, Python libs, cloud services |

## Output Documents

1. `<library>-api-overview.md` - High-level description
2. `<library>-authentication.md` - Auth setup and security
3. `<library>-endpoints.md` - Endpoint reference
4. `<library>-integration.md` - Integration guide
5. `<library>-config-template.md` - Configuration options
